Judul Proyek : Sistem Pengurutan Nilai Ujian Mahasiswa

Proyek ini merupakan sistem yang digunakan untuk mengurutkan data nilai ujian mahasiswa. sistem ini mengimplementasikan proses sorting bubble sort, pertama-tama sistem ini akan meminta input jumlah mahasiswa yang akan diurutkan nilainya kemudian meminta input nilai dari masing-masing mahasiswa, nilai tersebut akan diurutkan dengan sebuah fungsi bubble sort yang akan mengurutkan nilai mereka secara otomatis dan menampilkan nilai sebelum diurutkan dan setelah diurutkan.
